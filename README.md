# prototool 
proto 文件生成工具

# 命令
请查看help

```shell script
prototool -h
Usage:
  prototool [command]

Available Commands:
  config      生成配置文件
  fmt         格式化proto
  generate    编译proto
  help        Help about any command
  lint        对proto进行拼写检查

Flags:
  -h, --help   help for prototool

Use "prototool [command] --help" for more information about a command.
```
