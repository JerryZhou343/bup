# prototool 
proto 辅助工具，根据配置文件的依赖描述，编译，格式化proto，还可以对proto进行拼写检查

# 命令
请查看help

```shell script
prototool -h
Usage:
  prototool [command]

Available Commands:
  config      生成配置文件
  fmt         格式化proto
  generate    编译proto
  help        Help about any command
  lint        对proto进行拼写检查

Flags:
  -h, --help   help for prototool

Use "prototool [command] --help" for more information about a command.
```
